###UglyFoodFinder:
UglyFoodFinder helps you find markets that sell ugly produce. If you know of a market that sells ugly produce, add it on to this app.

###To install:

ruby-version: 2.3.0

```
bundle install
bundle exec rake db:create
bundle exec rake db:migrate
rails s
```
